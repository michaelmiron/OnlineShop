OnlineShop API Documentation

Base URL

http://localhost:5001/api

Products Endpoints

GET /products

Get all products

GET /products/:id

Get a specific product by ID

GET /products/search/:query

Search products by name, category, or description

POST /products

Create a new product

PUT /products/:id

Update a product

DELETE /products/:id

Delete a product

Orders Endpoints

GET /orders

Get all orders

GET /orders/:id

Get a specific order by ID

GET /orders/customer/:email

Get orders by customer email

POST /orders

Create a new order

PUT /orders/:id/status

Update order status

DELETE /orders/:id

Delete an order
